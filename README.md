# Simulating Pearl's 2009 SCM Paper

> **"Causal Inference in Statistics: An Overview"**  
> Judea Pearl (2009). *Statistics Surveys*, **3**, 96–146.  
> https://ftp.cs.ucla.edu/pub/stat_ser/r350-reprint.pdf

This repository implements and simulates the key concepts from Pearl's
foundational paper on Structural Causal Models (SCMs).

## Key Concepts Implemented

### 1. Structural Causal Models (SCMs)
A formal representation M = ⟨U, V, F, P(U)⟩ where structural equations
encode direct causal relationships as a DAG.

### 2. d-Separation
Algorithmic test for reading conditional independence from a DAG using the
**moral-graph** method (Pearl 2009, §2.3).  The three canonical structures
— fork, chain, collider — are demonstrated.

### 3. The do-Operator
`do(X = x)` sets variable X surgically, inducing the *mutilated model* M_x
where all incoming edges to X are removed.  This distinguishes
**interventional** P(Y | do(X)) from **observational** P(Y | X).

### 4. Backdoor Criterion and Adjustment (Pearl 2009, Theorem 3.3.2)

$$P(Y = y \mid \mathrm{do}(X = x)) = \sum_z P(Y = y \mid X = x, Z = z)\, P(Z = z)$$

Used to demonstrate **Simpson's Paradox**: a drug appears harmful in crude
data but beneficial when the health-status confounder is adjusted for.

### 5. Front-Door Criterion and Adjustment (Pearl 2009, Theorem 3.3.4)

$$P(Y = y \mid \mathrm{do}(X = x)) =
  \sum_m P(M = m \mid X = x) \sum_{x'} P(Y = y \mid M = m, X = x')\, P(X = x')$$

Applied to the **Smoking → Tar → Cancer** example where a genetic
confounder is unobserved but the mediator (tar deposits) is observable.

### 6. Mediation Analysis (Pearl 2009, §4)
Decomposition of total causal effect into **Natural Direct Effect** and
**Natural Indirect Effect** via path coefficients in a linear Gaussian SCM.
